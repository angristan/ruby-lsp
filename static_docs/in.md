### `in`

[Read this documentation in the browser](https://github.com/Shopify/ruby-lsp/blob/main/static_docs/in.md)

The `in` keyword is used in conjunction with `for` loops and pattern matching.

```ruby
for i in 1..5
  puts i
end
```
