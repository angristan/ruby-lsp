### `break`

[Read this documentation in the browser](https://github.com/Shopify/ruby-lsp/blob/main/static_docs/break.md)

The `break` keyword is used to exit a loop prematurely.

```ruby
i = 0
while i < 10
  break if i == 5
  puts i
  i += 1
end
```
