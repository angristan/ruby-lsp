### `return`

[Read this documentation in the browser](https://github.com/Shopify/ruby-lsp/blob/main/static_docs/return.md)

The `return` keyword is used to exit a method and optionally return a value.

```ruby
def add(a, b)
  return a + b
end
puts add(2, 3)
```
