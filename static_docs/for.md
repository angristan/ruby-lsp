### `for`

[Read this documentation in the browser](https://github.com/Shopify/ruby-lsp/blob/main/static_docs/for.md)

The `for` keyword is used to create a loop that iterates over a range or an enumerable object.

```ruby
for i in 1..5
  puts i
end
```
